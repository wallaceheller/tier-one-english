Aviso de Privacidad.

Privacidad amigable de principio a fin.

Introducción.

Tier One no recopila ningún tipo de información personal o confidencial de los usuarios. Respetamos su privacidad y estamos comprometidos con la transparencia en nuestras prácticas de datos.

Este sitio web no te "rastrea" y no tiene ningún interés en tus datos personales. Tomamos medidas cuidadosas para proteger tu privacidad. No se almacenan cookies ni datos personales cuando visitas este sitio web. Esto no significa que no se recopilen datos. Algunos servicios que utilizamos registran datos de acceso en sus servidores. Estos servicios son necesarios para garantizar que este sitio web se muestre en tu navegador como corresponde. Los siguientes párrafos recopilan toda la información relacionada. Si tienes alguna pregunta o sugerencia, no dudes en ponerte en contacto con nosotros.

GoatCounter.

Este sitio utiliza GoatCounter, una plataforma de código abierto que respeta la privacidad, para realizar análisis web sencillos. No se realiza un seguimiento del usuario con un identificador único y no se recopilan datos personales. Utilizamos los análisis para obtener información sobre cómo se utiliza este sitio web y para mejorarlo.

Páginas GitHub.

Este sitio web está alojado por GitHub Pages. No se proporcionan análisis al creador de este sitio. Este es un enlace a la Declaración de privacidad de GitHub.

Contacto via email.

Si escribes un correo electrónico a wallaceheller@tutamail.com, será manejado por Tuta Mail y tus datos personales quedan entonces sujetos a su declaración de privacidad.

Contacto via WhatsApp.

Si contactas al creador de este sitio por WhatsApp, tus datos personales estarán sujetos a su política de privacidad.

Privacidad de los niños.

Este sitio web no está destinado a niños y no recopila intencionalmente información de menores.

Usuarios internacionales.

El uso de GoatCounter exime a los creadores de este sitio del cumplimiento de las regulaciones internacionales de protección de datos como GDPR, CCPA o PIPEDA.

Cambios a este aviso.

El creador de este sitio web se reserva el derecho de actualizar este aviso de privacidad en cualquier momento sin previo aviso. El creador de este sitio web alienta a los usuarios a revisar este aviso periódicamente para conocer los cambios.

Contactar.

Si tienes alguna pregunta o inquietud sobre este aviso de privacidad, comunícate con el creador de este sitio web en wallaceheller@tutamail.com.

Descargo de responsabilidad.

Al acceder y utilizar este sitio web, tú reconoces haber leído, comprendido y aceptado este aviso de privacidad..
